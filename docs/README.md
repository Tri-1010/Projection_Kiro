## Tổng quan / Overview

Mô hình Markov-Chain dự báo rủi ro tín dụng với các tính năng:
- Chuỗi Markov không đồng nhất theo thời gian (time-inhomogeneous)
- Phân cấp phân khúc: FULL → COARSE → GLOBAL
- Shrinkage phân cấp (hierarchical shrinkage)
- Tail pooling cho MOB cao
- Hiệu chỉnh (calibration) theo đường cong DEL

Credit Risk Markov-Chain Projection Model with features:
- Time-inhomogeneous Markov chains
- Segment hierarchy: FULL → COARSE → GLOBAL
- Hierarchical shrinkage
- Tail pooling for high MOB
- DEL curve calibration
